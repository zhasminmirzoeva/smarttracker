# Smart Tracker
Мобильное кроссплатформенное приложение для отслеживания сроков годности продуктов питания. Ключевой функционал:
- Хранение информации о продуктах
- Автоматизированное доабвление продуктов по штрих-коду
- Напоминания о приближении/истечении срока годности
- Генерация ИИ-рецептов на основе имеющихся продуктов с учетом их срока годности, а также индивидуальных пожеланий пользователя


### Структура проекта

backend/app - исходные файлы серверной части проекта

mobile/src/features - исходные файлы мобильного приложения-клиентской части

tests - юнит-тесты


## Установка и запуск
Установка на Android

- Получить установочный файл [SmartTracker.apk.](https://disk.360.yandex.ru/")
- Передать APK-файл на мобильное устройство (через кабель, облачное хранилище или мессенджер).
- В настройках устройства при необходимости разрешить установку приложений из «неизвестных источников».
- Запустить файл SmartTracker.apk и выполнить установку.
- После завершения установки ярлык приложения появится на главном экране или в списке приложений.
- Запустить приложение и предоставить необходимые разрешения (уведомления, камера — по запросу системы).

Установка на iOS

Для iOS необходимо:

- Установить на устройство из App Store приложение Expo Go.
- Клонировать проект с GitHub:
`
git clone https://github.com/zhasminmirzoeva/smarttracker
cd mobile
`
- Установить зависимости проекта:
`
npm install
`
- Запустить проект:
`
npx expo start
`
- Дождаться запуска Expo DevTools и появления QR-кода в терминале или веб-интерфейсе. На iPhone открыть камеру и отсканировать код. Приложение загрузится и откроется в среде Expo Go.
